To uninstall SpatialShot on each operating system:

### Linux
1. Open a terminal.
2. Run the uninstaller script located in the application directory. By default, this is:
   ```bash
   ~/.local/share/spatialshot/app/uninstall.sh
   ```
3. Follow any on-screen instructions. You may need to manually remove the gsettings hotkey if you set one up, as prompted by the uninstaller.

### macOS
1. Open a terminal.
2. Run the uninstaller script located in the application support directory. By default, this is:
   ```bash
   ~/Library/Application Support/spatialshot/app/uninstall.sh
   ```
3. Follow any on-screen instructions. You may need to manually remove any configured hotkeys from 'System Settings' > 'Keyboard' > 'Keyboard Shortcuts...', as prompted by the uninstaller.

### Windows
You have two options to uninstall:

#### Option 1: Using the Uninstaller Executable
1. Navigate to the installation directory of SpatialShot (by default, `C:\Program Files\SpatialShot`).
2. Run `Uninstall.exe`.
3. Follow the prompts to complete the uninstallation.

#### Option 2: Using Windows 'Apps & features' (or 'Add/Remove Programs')
1. Open 'Settings' (Windows Key + I).
2. Go to 'Apps' > 'Apps & features' (or 'Control Panel' > 'Programs' > 'Programs and Features' for older Windows versions).
3. Find 'SpatialShot' in the list, click on it, and select 'Uninstall'.
4. Follow the prompts to complete the uninstallation.
